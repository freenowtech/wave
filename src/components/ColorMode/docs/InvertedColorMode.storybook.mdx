# InvertedColorMode

This renders a `div` tag with a specific classname based on the operating systems preferred color scheme.

The purpose is to be able to use the inverted variant compared to what the user has specified in their OS.
Change between light and dark on your `prefers-color-scheme` to see the effect on the button.
