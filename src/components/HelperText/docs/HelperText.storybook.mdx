# HelperText

This component is a support for Input, Textarea and Dropdown components to display error feedback
for an users action or to provide further information in advance.

-   HelperText could be shown either for a single or a cluster of sources (e.g. displaying an error message for two inputs)
-   Occurring error message will replace existing hint notifications
-   They are placed underneath its regarding source and they are left align
-   In case of multiple error notifications you should additionally show an error banner to provide global feedback

## Properties

<ArgTypes of={HelperTextStories} />

`HelperText` also supports all the properties of the [`Text`](?path=/docs/components-text--docs).

## Usage

### Provide additional information about form inputs

<Unstyled>
    <Input placeholder="Enter something" />
    <HelperText mt="1">127 Characters left</HelperText>
</Unstyled>

```jsx
<Input placeholder="Enter something" />
<HelperText mt="1">127 Characters left</HelperText>
```

### Validation messages

<Unstyled>
    <Input label="Email" placeholder="who@free-now.com" />
    <HelperText mt="1" variant="danger">
        Email is required
    </HelperText>
</Unstyled>

```jsx
<Input label='Email' placeholder="who@free-now.com" />
<HelperText mt="1" variant="danger">
    Email field is required
</HelperText>
```
